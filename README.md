# 🛡️ SOC Labs

## Overview

Welcome to my SOC Labs repository. This repository documents my hands-on learning in Security Operations Center (SOC) concepts, SIEM platforms, log analysis, threat detection, incident response, and blue team activities.

---

## Objectives

- Learn SOC operations

- Investigate security incidents
- Detect malicious activities
- Build detection rules
- Understand incident response workflows
- Prepare for SOC Analyst and Cyber Security Analyst roles
